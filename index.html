O R&M Dex é seu catálogo interdimensional para visualizar todos os personagens da série usando a API oficial.

Ver Personagens

Por que usar o R&M Dex?

Busca Dimensional

Encontre qualquer personagem por nome, status, espécie e mais.

Galeria Visual

Cards com imagens, status, localização e participação em episódios.

API Oficial

Conectado diretamente à API oficial Rick and Morty (https://rickandmortyapi.com).

Responsivo

Totalmente compatível com celulares, tablets e computadores.

Como Funciona

  1. Explore personagens: Acesse a galeria e veja informações completas de cada personagem.
  2. Use filtros: Busque por nome, status, espécie, origem ou localização.
  3. Amplie o universo: Descubra em quais episódios o personagem apareceu.
